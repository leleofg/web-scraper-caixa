**About**
---

A web scraper to access Caixa Economica Bank

**Installation**
---
`$ npm install`

**How To Use**
---

1. Run
    + `$ npm run start`
2. Open your browser
    + `$ localhost:4000`
3. Put login e password
4. See your account info

**Used libraries**
---

1. puppeteer
2. ejs
3. express

**Extracted data**
---

1. Balance
2. Statement simple
3. Statement full
4. Account
5. Branch
6. Operation
7. Name
