**About**
---

A web scraper to access Caixa Economica Bank

**Installation**
---
`$ npm install`

**How To Use**
---

1. Run
    + `$ npm run start`
2. Open your browser
    + `$ localhost:4000`
3. Put login e password
4. See your account info

**Used libraries**
---

1. puppeteer
2. ejs
3. express
